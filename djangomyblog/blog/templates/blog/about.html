Sobre o Meu Blog

Quem somos?

Somos da turma de "Fullstack com Python" do Senac Rio de Janeiro, na unidade de Campo Grande e este é nosso último projeto de aula.

Plataforma

Usamos Python com o framework Django, banco de dados SQLite e hospedado no serviço gratuito do PythonAnywhere.

Objetivos do site?

Experimentar o os recursos do Django e do Python e aproveitar para fazer um Blog funcional.

Além dos artigos postados pelos usuários cadastrados, temos comentários para cada artigo.
