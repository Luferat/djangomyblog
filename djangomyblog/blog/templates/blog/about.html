Sobre o Meu Blog

Quem somos?

Somos da turma de "Fullstack com Python" do Senac Rio de Janeiro, na unidade de Campo Grande e este é nosso último projeto de aula.

Plataforma

Usamos Python com o framework Django, banco de dados SQLite e hospedado no serviço gratuito do PythonAnywhere.

Objetivos do site?

Experimentar o os recursos do Django e do Python e aproveitar para fazer um Blog funcional.

Além dos artigos postados pelos usuários cadastrados, temos comentários para cada artigo.

Onde estão os códigos fonte?

Os códigos fonte deste aplicativo estão no GitHub.com.

Aproveite e faça um fork para que possa melhorar esse aplicativo, seja para seu uso ou para colaborar com o projeto.

Próximas etapas?

  • Melhorar a gestão dos usuários;
  • Permitir que somente usuários "autores" possam postar artigos;
  • Adicionar avatares para os usuários;
  • Adicionar thumbnais para os artigos;
  • Permitir formatação com "Markdown" nos comentários;
  • ...
